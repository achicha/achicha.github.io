Настраиваем Ubuntu после запуска.

Date 08 Ноябрь 2014 Category linux Tags linux

Google Drive

sudo add-apt-repository ppa:thefanclub/grive-tools
sudo apt-get update
sudo apt-get install grive-tools
#в поиске набираем Grive setup и следуем инструкциям

Отключаем комерческий сервис в меню Dash

 gsettings set com.canonical.Unity.Lenses disabled-scopes "['more_suggestions-amazon.scope', 'more_suggestions-u1ms.scope', 'more_suggestions-populartracks.scope', 'music-musicstore.scope', 'more_suggestions-ebay.scope', 'more_suggestions-ubuntushop.scope', 'more_suggestions-skimlinks.scope']"

Разблокировать dpkg

Часто появляется ошибка: Unable to lock the administration directory (/var/lib/dpkg/) is another process using it

ps -A | grep apt-get
sudo kill -9 <process-id>
sudo kill -9 2345

Создание загрузочной флешки из образа диска

sudo dd if=/home/username/path/filename.iso of=/dev/sdb bs=16M
  1. ставить на форматированную fat32 флешку.
  2. включить галку - загрузочный диск.
  3. ставлю на sdb раздел, на sdb1 не получается.
  4. bs=16m сколько за раз записывается.

Как удалить приложение со всеми зависимостями

sudo apt-get remove firefox* # звездочку добавляем

VLC

sudo add-apt-repository ppa:djcj/vlc-stable
sudo apt-get update && sudo apt-get install vlc

Добавляем ключ от репозитория (как пример)

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys FC91AE7E 
apt-key list
