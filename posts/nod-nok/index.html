Нахождение НОД и НОК.

Нахождение НОД

Наибольший общий делитель - самое большое число, на которое можно без остатка разделить два числа, для которых ищется НОД.

Алгоритм:

  1. Большее число делим на меньшее.
  2. Если делится без остатка, то меньшее число и есть НОД (следует выйти из цикла).
  3. Если есть остаток, то большее число заменяем на остаток от деления.
  4. Переходим к пункту 1

Код

a = 12
b = 16
while a!=0 and b!=0:
    if a > b:
        a = a % b
    else:
        b = b % a
print (a+b)

Нахождение НОК

Наименьшее общее кратное - наименьшее число, которое делиться без остатка на каждое из двух заданных числел.

Алгоритм

НОК = a*b / НОД(a,b)

Код

a = 12
b = 16
#другой способ нахождения НОДа
def gcd(a, b):
    return b and gcd(b, a%b) or a
#выводим НОК
print ((a*b)/gcd(a,b))
