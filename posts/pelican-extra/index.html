Pelican. Расширенные настройки.

Date 01 Ноябрь 2014 Category web Tags pelican / python

Для начала необходимо выбрать хорошую тему, в которой предустановлено большинство нужных вещей, а потом уже допиливать её по необходимости :)

Все изменения вносятся в основной файл настроек pelicanconf.py

Структуризация файлов.

Складываем все посты в отдельную папку.

OUTPUT_PATH = 'output/articles/'
ARTICLE_URL = 'articles/{slug}'
ARTICLE_SAVE_AS = 'articles/{slug}/index.html'

BREADCRUMBS

Или хлебные крошки - для улучшения навигации по сайту.

DISPLAY_BREADCRUMBS = True
DISPLAY_CATEGORY_IN_BREADCRUMBS = True

Pygments для подсветки кода в блоках.

#устанавливаем pygments и настраиваем, если не установлено по дефолту
PYGMENTS_STYLE = 'zenburn'

Favicon - иконка для сайта.

Копируем картинку .png или .ico в наш проект и настраиваем

FAVICON = 'images/favicon.ico'

Robots.txt

Для лучшей индексации сайта роботами, так же можно запретить индекс различный файлов. Необходимо создать robots.txt в корне папки output и прописать необходимые настройки:

User-agent: Yandex
Disallow: /content/

User-agent: *
Disallow: /content/
Sitemap: http://site_name/sitemap.xml
Host: site_name

Облако тегов

# отображение тегов
DISPLAY_TAGS_ON_SIDEBAR = True
# показывать в облаке или списком
DISPLAY_TAGS_INLINE = True

Sitemap

Для улучшения индексации сайта в поисковиках. XML карта сайта

1) скачиваем плагин sitemap

2) устанавливаем в папку plugins

3) прописываем настройки:

# Plugins
PLUGIN_PATH = "plugins"
PLUGINS = ["sitemap", ]
# Sitemap
SITEMAP = {
    'format': 'xml',
    'priorities': {
        'articles': 0.5,
        'indexes': 0.5,
        'pages': 0.5
    },
    'changefreqs': {
        'articles': 'monthly',
        'indexes': 'daily',
        'pages': 'monthly'
    }
}

4) читаем мануал по настройкам
